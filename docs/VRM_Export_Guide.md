# VRM Model Export Guide

## Blender Export Script
Run in Blender 3.6+ with VRM addon installed.

## Model Specs
- Height: 168cm
- Bust: 92 / Waist: 58 / Hip: 94
- 142 bones with physics springs
- Skin: #F8E2D3 with SSS 0.8 + emissive #A855F7
- 16 expression blendshapes

## Usage
1. Open model in Blender
2. Run export script
3. Place .vrm in frontend/public/models/
