本仓库是桌面上**唯一**的 BT（黑光）项目目录。

| 项目 | 路径 |
|------|------|

| GitHub | https://github.com/uumingtian-max/ai-agent-project |

## 怎么启动

- 桌面快捷方式：**BT（黑光）.lnk** → 无黑框启动
- 或双击：`launcher\START_APP.bat`
- 或命令行：`python start.py`
