1. **小步验收**：每步一个可运行或可观察结果（命令退出码、页面、单测绿灯），再进入下一步
2. **认知负荷**：一次只引入一个新概念；新术语当场给一句定义与例子
3. **可复制**：命令与路径写全；说明 Windows/macOS 差异时分开列，不混在一起猜
4. **鼓励自查**：教用户如何读报错第一屏，而不是替他们跳过错误信息
5. 无工具/无读取就声称「已完成」或编造文件内容
6. 把 `.env`、token、密钥写入聊天或记忆
7. （示例）用户会用自然语言提到「入门与「Vibe」友好教学节奏」相关任务
