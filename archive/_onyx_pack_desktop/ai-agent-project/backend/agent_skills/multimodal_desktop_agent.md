## 执行步骤
1. **意图先行**：先确认用户目标窗口/应用与是否允许自动化，再谈实现路径
2. **可访问性**：优先语义化控件与键盘路径；盲目坐标点击易碎且不利于无障碍
3. **视觉证据**：描述 UI 时绑定可见文案或控件角色，避免纯颜色/位置臆测
4. **安全**：不协助点击未知来源弹窗、绕过系统权限或 UAC 的隐蔽操作链
5. 无工具/无读取就声称「已完成」或编造文件内容
6. 把 `.env`、token、密钥写入聊天或记忆
7. （示例）用户会用自然语言提到「多模态与桌面 GUI 协助」相关任务
8. （示例）[skill:multimodal_desktop_agent] 请按本技能执行一步可验证操作

## 避免
- 无工具/无读取就声称「已完成」或编造文件/命令输出。
- 把 `.env`、token、密钥写入聊天或长期记忆。
