# 白月光 · 清贵质 — 终极整合 VRM 查看器

## 五份合一 · 全功能集成

| 来源 | 贡献 |
|------|------|
| 第一份（CDN修复） | CDN 切换 jsdelivr→unpkg，消除加载失败 |
| 第二份（整合版） | 首饰/鞋子切换，动作增强 |
| 第三份（建模代码） | SSS皮肤shader，布料特效，建模规范 |
| 第四份（灵魂补全） | 呼吸/心跳/眼神漂移/微表情/物理/视线跟随/点击反馈 |
| 第五份（究极满配） | 24套衣服 + 8款发型 + 材质特效大满贯 |

## 已集成功能

- ✅ **24套衣服**：基础6 + 进阶6 + 特效6 + 定制6（每套带专属shader）
- ✅ **8款发型**：Kajiya-Kay 双高光 + 流光 + 边缘透光
- ✅ **4套首饰**：细链/小珠/细圈（.vrm 子模型加载）
- ✅ **3款鞋子**：小羊皮/细带/软布
- ✅ **8种姿态**：静立/侧转/垂眸/抬眼/回眸/柔婉/端庄/柔荑
- ✅ **皮肤SSS**：三层次表面散射 + 绒毛高光
- ✅ **动态生命**：呼吸（4s周期）+ 心跳（0.8s）+ 眼神漂移 + 微表情
- ✅ **交互**：视线跟随鼠标 + 距离冷暖调 + 点击触碰反馈
- ✅ **光影**：四点布光 + 软阴影 + 雾效
- ✅ **声音**：Howler.js 清柔声线

## 使用方式

1. 将 `pure_noble.vrm` 放入 `asset/` 目录
2. 将对应贴图放入 `asset/cloth/`、`asset/hair/`、`asset/shoe/`、`asset/skin/`、`asset/eye/`
3. 将首饰 VRM 文件放入 `asset/acc/`
4. 将声音文件放入 `asset/voice/soft_clear.mp3`
5. 双击 `index.html` 或浏览器打开

## 文件结构

```
白月光-VRM/
├─ index.html          ← 主文件（五份合一，20KB）
└─ asset/
   ├─ pure_noble.vrm   ← 核心模型（VRM 1.0，无中文名）
   ├─ cloth/           ← 24套衣服贴图（PNG24）
   ├─ hair/            ← 8款发型贴图
   ├─ shoe/            ← 3款鞋子贴图
   ├─ acc/             ← 4套首饰子模型（.vrm）
   ├─ skin/            ← 皮肤贴图（color/normal/rough）
   ├─ eye/             ← 眼睛贴图（iris/normal）
   └─ voice/           ← 声音文件（MP3 44100Hz 128kbps）
```
