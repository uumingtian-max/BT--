# 接口清单

## 认证
- `POST /auth/register` 用户实名注册
- `POST /auth/login` 登录

## 后台
- `GET /admin/users` 用户列表
- `POST /admin/managers` 创建管理账号
- `POST /admin/recharge` 充值余额
- `POST /admin/status` 封禁/解封账号
- `POST /admin/mute` 禁言
- `POST /admin/notices` 全体公告/单人通知
- `POST /admin/ip-blacklist` IP 拉黑
- `POST /admin/device-ban` 设备封禁
- `POST /admin/backup` 数据备份
- `GET /admin/audits` 审计日志

## 聊天
- `POST /chat/messages` 发消息
- `GET /chat/messages?room=public` 读消息

## 红包
- `POST /red-packets` 发红包
- `POST /red-packets/{packet_id}/claim` 抢红包

## 通知
- `GET /notices` 获取公告/通知
- `POST /push/simulate` 模拟推送

## 高级能力
- `POST /speech/transcribe` 模拟语音转文字
- `POST /video/rooms` 创建模拟视频房间 token

## 健康检查
- `GET /health`
