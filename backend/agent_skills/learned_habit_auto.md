## 当前行为模式
- 近期窗口与进程显示你在处理模型、训练或推理相关资产。
- 近期活动显示你在反复调试部署、服务启动或前后端联调。
- 你在多种编程 Agent / IDE 助手之间切换，属于高密度开发节奏。
- 你在整理桌面、资产清单或工作区文档类材料。
- 保持输出：中文、先结论、短段落、可执行下一步。
- 活跃高峰约在 15 点（本地时区）。
- 近一周工具调用总体成功率约 91.8%，成功 338 / 失败 30。
- 以下工具近期失败偏多：strong_search、write_file

## 建议执行调整
- 涉及模型与权重时，先盘点本地目录与显存占用，再给出可执行整理方案。
- 部署类问题优先查端口、日志与依赖版本，再给最小变更修复路径。
- 本地 Agent 任务优先走工具链拿证据，避免与 IDE 内助手重复空转。
- 文件类任务先列目录再读关键文件，输出结构化清单。
- 长任务拆成可验证小步，每步对应工具或命令级产出。
- 在高峰时段优先安排需要交互与编译的重活。
- 对这些工具先缩小输入规模或换路径验证，再扩大任务范围。
